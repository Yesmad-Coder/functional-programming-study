## 목표

1. 함수형 프로그래밍 **이론 이해**하기
2. 익숙하지 않은 단어나 예시는 반복 작성한 적이 있더라도 그때그때 찾아 기록하기
3. 예제 코드를 작성할 때는 천천히 **이해하며** 작성하기
4. 습득한 이론으로 개발할 것 생각하기
5. 생각만 하지말고 **실천**으로 이어보기
6. 자바스크립트를 보다 깊게 이해하기
7. 어려워도 끝까지 해보기 🙏🏼
