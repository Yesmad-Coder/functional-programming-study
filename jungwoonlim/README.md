# 자바스크립트 함수형 프로그래밍 스터디

> 함수형 프로그래밍의 이론을 이해하고 생각하며 코드 작성해보자

## 목표

1. 함수형 프로그래밍 **이론 이해**하기
2. 예제 코드를 작성하더라도 **생각하며 코드 작성**하기
3. 습득한 이론으로 개발할 것 생각하기
4. 생각만 하지말고 **실천**으로 이어보기
5. 자바스크립트 예시를 타입스크립트로 작성해보기
